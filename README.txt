

productList generates a list of products from amazon with a given search term. It returns a list of products with the following information: title, price, rating, number of reviews, url, image url, and a list of features. The list of features is a list of strings, where each string is a feature of the product. The list of features is not always available.

sortProduct sorts a list of products by a given attribute. The attribute can be one of the following: title, price, rating, or number of reviews. The sortProduct function returns a list of products sorted by the given attribute. also contains specific functions for obtaining more information about a specific amazon product. including shipping information.
